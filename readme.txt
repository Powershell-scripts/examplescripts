 Репозиторий только для примеров работы с теми или иными проблемами, на решение которых ушло более 30 минут.
 
 Тут будет целая куча разношерстных скриптов предназначенных ТОЛЬКО в качестве примеров. Это не автоматизация чего либо, посему и написано на скорую руку. 
 
 *** ВАЖНО ***
 Все примеры проверены только на PS 5.1 
 
На данный момент тут находятся примеры:
1. ExampleWorkingLongPath - пример показывающий на основе move-item как использовать длинные пути в OS windows (простая концепция, но времени потратил  примерно 1ч на реализацию. 
	1.1 Из за того, что Windows исторически работает только с длинной пути до 260 символов - эта проблема имеет место быть. Как путь решения - использовать встроенный во многие командлеты параметр literalpath 
	1.2 Для корректного отображения пути - советую использовать конструкцию в виде (как в скрипте) $variable.fullname для вызова свойств объекта (тогда пути будут выводится в корректном формате) 
2. Func_Translate - Функция преобразования кириллицы в латиницу. Долго мучился, что бы корректно написать массив. Можно пользоваться, но для нужд AD еще надо дописать форму для забора и данных от куда нибудь. Как пример можно рассмотреть мой репозиторий CreateUserADT1 - там мы берем данные из excel. 
	2.1 А так же имеет смысл сменить read-host на чтение из переменной. 
3. FuncGetUserAccountInfo - Недоделанная функция получения данных пользователя из AD 
4. WebBasicAuthorisation - Пример базовой аутентификации на сайте с помощью Powershell 
5. WebOAuth - Пример работы с Rest API сайтов. Брал за пример Pyrus (https://pyrus.com/ru/help/api) Мне требуется разобраться со скачиванием файлов. Подготовил пару переменных для нее. В процессе.
